# Dashboard Design Placeholder

- Watchlist Summary
- Watchlist Detail
- Symbol Dashboard
- Trigger Line Dashboard
- Breakout Event Dashboard
- Paper Trading Dashboard
- Paper Trading Settings

The design should support:

- Multiple trigger lines per symbol
- Historical and active line states
- Swing analysis context
- Breakout and breakdown history
- Paper-trade outcomes and performance analytics
- Future real-order linkage without redesigning the data model
